依存パッケージ

dependencies

ポイント

  • ステート管理は、riverpod を利用しています。
  • ルーティングは、go_routerを利用しています。
  • ビデオ通話は、agora_rtc_engineを利用しています。
  • immutableなステートを実現するために、freezedを利用しています。
  • 機密性の高いconfigを外出しするために、flutter_dotenvを利用しています。
  • アプリ停止しても永続化しておきたいステートを保持するために、shared_preferencesを利用しています。

dev_dependencies

ポイント

  • テストは、mocktail を利用しています。
  • goldent_test と integration_test も利用しています。
  • テストの実装状況ですが、channel、user ディレクトリの部分で、Firestoreへデータを出し入れする箇所のUnitテストは実装しました。 しかし、最適化・最新化する等、まだ十分に実装できていない状況です。また、その他のテストはまだ未実装です。
